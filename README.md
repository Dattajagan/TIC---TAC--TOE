# Tic Tac Toe

A modern, interactive Tic Tac Toe game with two-player and single-player (CPU) modes, built with vanilla JavaScript.

## Features

✨ **Two Game Modes**
- 🎮 **Two Players** — Play against a friend on the same device
- 🤖 **Single Player** — Challenge an intelligent CPU opponent with strategic gameplay

🎨 **Modern UI**
- Glassmorphic design with smooth animations
- Responsive layout that works on desktop, tablet, and mobile
- Glow effects and polished interactions
- Winner highlighting with animated cells

📊 **Score Tracking**
- Persistent scoreboard for X, O, and draws
- Real-time score updates

⚡ **Smooth Gameplay**
- Instant feedback on cell clicks
- CPU thinking animation
- Draw detection and game-over messaging

## How to Play

1. Select a game mode: **Two Players** or **Single Player (vs CPU)**
2. Click cells to place your mark (X goes first)
3. Get three marks in a row (horizontally, vertically, or diagonally) to win
4. Click **New Game** to reset and play again
5. Scores are tracked across multiple games
